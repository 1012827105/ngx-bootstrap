Usage

Examples

String items:

All items could be freely dragged between sortable containers with the same dropZoneGroup property

One way drag:

By default sortable container rejects any external item but it is possible to allow to drag items from specific containers using acceptFromZones property

Complex data model:

Normally sortable container displays items as strings but it is also possible to display value of specific item's property using fieldName property

Custom item template:

Often it is necessary to specify more complex template for the draggable items you can do it using itemTemplate property

Directives usage:

Also you can add a couple of directives to your table or another collection to make it sortable. For example it is the only way to make table sortable since it is impossible to do via custom template above.
